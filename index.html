Projek dan Pengalaman

Proyek pertama saya adalah sebuah game RPG sederhana yang saya buat menggunakan Python. Game ini merupakan langkah awal saya dalam memahami logika pemrograman dan interaksi pengguna. Sistem dalam game ini dirancang menggunakan menu interaktif berbasis input numerik, di mana pemain dapat mengetik angka seperti 1, 2, 3, atau 0 untuk kembali ke menu sebelumnya. Proyek ini mengajarkan saya cara membuat antarmuka terminal yang fungsional, sekaligus memberikan pengalaman langsung dalam mengembangkan alur permainan yang menarik. Selain itu, saya juga berkontribusi dalam sebuah proyek kolaboratif untuk membuat aplikasi toko buku berbasis terminal menggunakan Python. Aplikasi ini bertujuan untuk mengelola data buku secara sederhana, termasuk memasukkan, melihat, dan menghapus data. Dalam proyek ini, saya belajar tentang konsep CRUD (Create, Read, Update, Delete) dan pentingnya kerja sama tim dalam menyelesaikan proyek berbasis teknologi. Pengalaman ini juga membantu saya memahami alur pengembangan aplikasi dari awal hingga selesai.
